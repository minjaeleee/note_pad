<h1> 실행 방법 </h1>

<h3> Front-end 실행하기  </h3>
client 폴더로 이동
<pre> cd client </pre>
package install
<pre> yarn install  </pre>
start
<pre> yarn start  </pre>

<h3> Back-end 실행하기  </h3>
server 폴더로 이동
<pre> cd server </pre>
package install
<pre> yarn install  </pre>
start
<pre> yarn start  </pre>
